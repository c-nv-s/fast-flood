Welcome to Fast Flood

These are the rules of the game:

  1. You win if you make all the squares in the grid the same color.
  2. You may receive (imaginary) medals if you solve the puzzle as fast as you can and with the fewest possible moves.
  3. You start at the square on the top-left corner and change the color of the adjacent squares by clicking on the button with the same color.
  4. You have a limited number of moves and time. If you go over any of them, you lose.
  5. You can try 3 times in a row per day to solve the puzzle. There's a new puzzle every day.
